#Sales Analysis Dashboard
Overview
This Sales Analysis Dashboard provides comprehensive insights into sales performance, product trends, customer behavior, and profitability metrics. The dashboard is designed to help stakeholders make data-driven decisions by visualizing key business metrics across multiple dimensions.
Dashboard Structure
Main Navigation
The dashboard consists of five main sections accessible via the left sidebar:

Sales - Overall sales performance and trends
Category - Category-wise analysis and breakdowns
Customer - Customer segmentation and behavior
Product - Product performance metrics
Profit - Profitability analysis

Filters and Interactivity
Global Filters (Available on all pages):

Year - Filter data by specific year(s)
Category - Filter by Technology, Furniture, or Office Supplies
Region - Geographic region selector

Interactive Features:

Click on chart elements to filter related visualizations
Hover over data points for detailed tooltips
Drill-down capability in hierarchical charts
Cross-filtering across multiple visualizations
